Outline

Creating a Tic-Tac-Toe game:

  1. Print the board before each turn
  2. Accept user input
  3. Validate user move
  4. Evaluate possible moves
  5. Choose computer move
  6. Test for Win! condition

Creating a mini platformer adventure game:

  1. Make several maps using a free to use game engine.
  2. Sending it to the team for assessment and troubleshooting.
  3. Create several characters and assign keys to actions.
  4. Create the npcs and their auto functions based on character actions, anticipate every possible scenario and program it.
  5. Set win conditions and troubleshoot the full game part by part until completion.
